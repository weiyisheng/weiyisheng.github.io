UnsupervisedLearning

无监督学习（Unsupervised learning）

Unsupervised learning allows us to approach problems with little or no idea what our results should look like. We can derive structure from data where we don’t necessarily know the effect of the variables.

We can derive this structure by clustering the data based on relationships among the variables in the data.

With unsupervised learning there is no feedback based on the prediction results.

从数据集中提取特征

We are given the data set and we are not told what to do with it and we are not told what each data point is. Instead we are told, here is the data set, and can you find some structure in the data?

SupervisedLearning

监督学习（Supervised learning）

In supervised learning, we are given a data set and already know what our correct output should look like, having the idea that there is a relationship between the input and the output.

根据数据集中明确的结果（“correct output”），算法能够对未来的输入进行输出的预测。

我的理解，因为训练数据集中有明确的结果，输入输出有明确的对应关系，是为“监督”一词。

分类

1. regression problems

we are trying to predict results within a continuous output, meaning that we are trying to map input variables to some continuous function.

连续性

根据已有的size和prise的数据集，得出一个size-prise的曲线，能够对prise进行预测。

2. classification problems

we are instead trying to predict results in a discrete output. In other words, we are trying to map input variables into discrete categories.
